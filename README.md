# Createch - Frontend

Este repositorio contiene el código fuente del frontend del sitio web principal de Createch, desarrollado utilizando **Next.js** y **React**.

## Requisitos Previos

Asegúrate de tener instalado en tu sistema:
- [Node.js](https://nodejs.org/) (Versión 18.x o superior recomendada)
- npm (viene incluido con Node.js)

## Instalación paso a paso

1. **Clonar el repositorio:**
   ```bash
   git clone https://github.com/TU_USUARIO/createch-web.git
   cd createch-web
   ```

2. **Instalar las dependencias:**
   Dentro de la carpeta del proyecto, ejecuta el siguiente comando para descargar e instalar todas las librerías necesarias:
   ```bash
   npm install
   ```

## Ejecutar el proyecto en desarrollo

Una vez instaladas las dependencias, puedes levantar el servidor de desarrollo local ejecutando:

```bash
npm run dev
```

El sitio web estará disponible y ejecutándose en [http://localhost:3000](http://localhost:3000). 
Cualquier cambio que realices en el código se actualizará automáticamente en el navegador.
